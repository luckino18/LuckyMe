# LuckyMe Seeker 1.2.2 — Release Notes

## Solana Mobile Publisher — What's New

LuckyMe 1.2.2 completes the redesigned Seeker experience. Paid-pool confirmations now remain inside the new visual theme, startup stays branded while verified mainnet data loads, and Activity restores transparent public round history with winners, SOL prizes, automatic refunds and direct Solscan verification. The update also retains the four redesigned pools, Referral League, NFT Holders campaign, Latest Winners and smoother in-app navigation.
