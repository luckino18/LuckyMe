# Security Policy

LuckyMe is an experimental Solana devnet MVP. Do not use it with mainnet funds.

## Current Status

- The program has not been externally audited.
- The randomness design is an MVP commit-reveal flow and must be hardened before production.
- The economic model includes gambling-like mechanics and requires legal review before any real-money launch.

## Reporting Issues

Open a GitHub issue for non-sensitive bugs.

For sensitive findings, do not publish exploit details publicly until a private reporting path is configured.
