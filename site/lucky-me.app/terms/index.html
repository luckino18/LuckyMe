Service

LuckyMe is a Solana mobile application for fixed-entry pool rounds. Users connect a compatible Solana wallet, review transaction details, and decide whether to sign transactions in their own wallet.

Wallet Control

LuckyMe does not ask for seed phrases or private keys. The backend prepares unsigned transactions. Your wallet controls signing, approval, rejection, and fee payment.

Network And Transaction Risk

Solana transactions are irreversible once confirmed. Network fees, congestion, RPC availability, wallet behavior, and on-chain program state can affect transaction outcomes.

Availability

LuckyMe may show unavailable states when backend, RPC, wallet, or on-chain data cannot be verified. Production does not use fake pool data as a replacement for mainnet state.

Support

For support, use the contact route at lucky-me.app/support.
