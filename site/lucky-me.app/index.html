Solana mainnet pools

Mobile-first luck pools on Solana.

LuckyMe lets users connect a Solana wallet, review a fixed-entry pool ticket, and sign the transaction in their wallet. The backend builds unsigned transactions and never signs for players.

Contact support View API status

Mainnet Ready

The production target is Solana mainnet-beta with wallet chain validation and HTTPS backend validation.

User-Signed

Player transactions are reviewed in-app and signed by the connected wallet. LuckyMe does not custody user private keys.

Transparent Rounds

Pool pricing, round state, treasury split, jackpot contribution, and program details are shown before signing.
